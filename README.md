# GitHubユーザー検索アプリ

GitHubのユーザー名を検索すると、プロフィール情報を取得して表示するReactアプリです。

## 🔧 使用技術

- React + Vite
- Tailwind CSS
- GitHub REST API
- React Router

## 🚀 機能

- GitHubユーザーのプロフィール表示
- 検索履歴の表示・クリックで再検索
- ページ遷移による詳細表示
- 履歴のローカル保存（localStorage）
- 検索履歴のリセットボタン付き
